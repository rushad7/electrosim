# electrosim
Python package for electronics simulations.It can be used to simulate The following-

1. Direct Current
2. Alternate Current
3. Resistor
4. Capacitor
5. Inductor
6. Mesh Circuits
7. Impedance
8. AND Gates
9. NAND Gates
10.OR Gates
11.NOT Gates
12.XOR Gates
13.NOR Gates
14.XNOR Gates


# Required Packages
1. Numpy - Command "pip install numpy"
2. sympy - Command "pip install sympy"
